    products: []
    description: >-
      Added publicly available REST endpoints for fetching information and data
      of shared files/directories. The data-related endpoints are offered by
      Onezone, which redirects to a suitable Oneprovider so that a guest user
      does not need any knowledge of the environment to access the data.
      Improved the Web GUI's shares view to present the public endpoints in an
      easy-to-use manner.
  - issue: VFS-7280
    products: []
    description: >-
      Fixed page reload after Let's Encrypt certificate generation via Onepanel
      GUI.
  - issue: VFS-7276
    products: []
    description: >-

      handling of various exceptions and more informative error messages
      including Oneclient and Oneprovider compatibility and invalid token
      issues.
  - issue: VFS-7275
    products: []
    description: >-
      Improved Oneclient exception handling, including connection errors and
      invalid tokens.
  - issue: VFS-7274
    products: []
    description: >-
      Added SIGTERM and SIGINT handlers to Oneclient, ensuring that after the
      oneclient process is stopped by some other process, the mountpoint is
